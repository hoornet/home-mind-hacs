## Home Mind

A Home Assistant conversation agent powered by Claude AI with persistent memory.

### Features

- Natural language control of your smart home
- Persistent memory that learns your preferences
- Semantic search with cognitive memory
- Multi-turn conversation support
- Real-time sensor data queries

### Requirements

This integration requires a running [Home Mind API server](https://github.com/hoornet/home-mind).

### Quick Start

1. Install and run the Home Mind API server
2. Install this integration via HACS
3. Add the integration in Home Assistant settings
4. Configure your API server URL
5. Start using natural language to control your home!
